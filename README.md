# Policy Portal

An auto insurance portal that allows users to view policy details, initiate claims, and track claim status.

## Getting Started

1. Install dependencies for both the client and server:

```
npm install
cd client && npm install
cd ../server && npm install
```

2. Start the development server:

```
npm start
```

This will concurrently start the React development server and the Express.js API server.

## Project Structure

- `client/`: React frontend application
- `server/`: Express.js backend API
